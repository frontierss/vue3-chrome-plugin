# vue3 + vite 製作 chrome plugin

利用vue3 + vite製作chrome 擴充, 介面部分利用tailwind CSS。

## Project setup
```
npm i
```

## Compiles and hot-reloads for development
```
npm run dev
```

## Compiles and minifies for production
```
npm run build
```

## 如何預覽?
```
執行npm run build後, 在chrome > 擴充功能 > 按下[載入未封裝項目] 選擇dist資料夾即可預覽
```
